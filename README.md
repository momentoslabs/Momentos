# Momentos

**Take back your browser's start page.**

Momentos is a customizable, modular browser start page that puts you in control. Create, customize, and arrange interactive widgets (modules) to build the perfect portal for your daily browsing.

## Features

- **Modular Widget System**: Add weather, news, clock, and custom modules to your grid
- **Drag & Drop Interface**: Arrange modules exactly how you want them on an 8×4 grid
- **Search Integration**: Quick access to Google, Bing, and social media like Instagram
- **Full Customization**:
  - 7-color theme system with automatic light/dark detection
  - Color presets (Dark, Light, or create your own)
  - Custom module support via URL
  - Import/Export layouts as .mmt files
- **Module Marketplace**: Browse and add verified modules, or paste your own custom module URLs
- **Responsive Design**: Clean, modern interface with dynamic typography

## Usage

### Adding Modules

1. Click the **Modules** button (☰) to open the drawer
2. Browse verified modules in the **Search** tab
3. Click **+ Add** to add modules to your collection
4. Drag modules from **My Modules** onto the grid

### Custom Modules

Paste any module URL in the **Search** tab to add custom modules. Unverified modules will show a warning.

### Customizing Colors

1. Click the **Settings** button (⚙️) in the top bar
2. Go to the **Interface** tab
3. Use color presets or customize all 7 colors individually
4. Theme automatically adjusts text color based on luminance

### Import/Export Layouts

Save and share your layouts:

1. Open the Modules drawer
2. Click the **⋮** menu button
3. Choose **Export .mmt** to save or **Import .mmt** to load a layout
